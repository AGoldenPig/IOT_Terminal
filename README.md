## 版本号说明

A.B.C.D.E

- A表示需求版本号
- B表示一级功能版本号（反映功能编写进度，某功能需求版本）
- C表示功能内部模块版本号（反映功能内部模块编写进度）
- D表示功能内部模块某框架版本号（反映功能内部模块的框架编写进度）
- E表示功能内部模块某框架内容版本号（反映功能内部模块的框架的内容编写进度与测试进度）
